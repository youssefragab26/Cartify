# 🛒 Cartify – Modern E-Commerce Web App

Cartify is a modern e-commerce web application built with **Next.js, TypeScript, and Tailwind CSS**.  
It allows users to browse products, add them to a cart, and manage orders in a clean and responsive UI.

---

## 🚀 Features
- 📦 Product listing & product details
- 🛒 Add / remove items from the cart
- 📑 Order summary & checkout flow
- 📱 Fully responsive design (mobile-first)
- ⚡ Type safety with TypeScript
- 🎨 Styled with Tailwind CSS
- 🔗 Routing with Next.js App Router

---

## 🛠️ Tech Stack
- **Next.js** – React framework  
- **TypeScript** – Type safety  
- **Tailwind CSS** – Styling  
- **Next.js App Router** – Navigation & routing  
- **ESLint & Prettier** – Code quality
